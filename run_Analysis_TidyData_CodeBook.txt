Data Dictionary - Combined data set representing data from both Train and Test extracts alongwith Activity and Subject.
"Activity"
	Description: Defines activities performed by volunteers
	Data Type: Character String
	Values: WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
"Subject"
	Description: Categorizes volunteers into different buckets based on the activities they performe
	Data Type: Integer
	Values: 1 to 30
"TrainorTestIndicator"
	Description: Identifies the extract from which data was originally pulled.
	Data Type: character string
	Values: TRAIN, TEST
